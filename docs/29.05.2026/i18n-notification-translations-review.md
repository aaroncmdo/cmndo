# i18n Notification-Übersetzungen — Native-Review-Sheet

Platzhalter `{1}`,`{2}`… = eingesetzte Variablen (Name, Datum, Link, Betrag …) — nicht übersetzen, Position kann sprachbedingt wandern. Verbatim bleiben: **Claimondo, LexDrive, WhatsApp, URLs, EUR, §249 BGB**.
