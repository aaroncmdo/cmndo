## Harte Regeln + Gotchas (gelernt/bestätigt)

1. **Kanzlei-Namensregel:** Partner-Kanzlei (LexDrive) auf Marketing-/Rezitations-Flächen NIE namentlich → „unsere Partnerkanzlei". Operativer Kundenkontakt (Kunde-Portal, Magic-Link, `/flow/[token]`, `MeineKanzleiCard`) + intern OK. Memory `feedback_kanzlei_nie_namentlich`.
2. **Neue public Content-Route MUSS** in `src/lib/supabase/middleware.ts` `publicPaths` UND `src/proxy.ts` `MARKETING_PREFIXES` — sonst 307→/login für anon+Crawler, **Build bleibt grün** (stiller Trap). `dynamicParams = false` für harten 404. (Stream C-Rest brauchte das NICHT — keine neue Route, nur Komponente in bestehende.)
3. **Build:** `NODE_OPTIONS=--max-old-space-size=8192` (TS-Check-OOM). `next start` kaputt mit `output: standalone` → mit `next dev` smoken. Lokaler `next build` kann an `/gutachter-partner` Export-Timeout scheitern = Parallel-Session-Prod-DB-Contention, NICHT der eigene Code. (Bei Stream C-Rest lief der volle Build diesmal sauber grün durch.)
4. **Worktree-Setup:** Git-Worktree hat kein `node_modules` → Junction auf Haupt-Repo: `New-Item -ItemType Junction node_modules -Target <main>\node_modules`. Für Build zusätzlich `.env.local` aus Haupt-Repo kopieren. (`node scripts/new-session-worktree.mjs <slug> staging` legt den Worktree an, macht aber Junction/env NICHT — manuell nachziehen.)
5. **Commit-Messages:** über Bash-Tool (git bash) per Heredoc-Datei + `git commit -F` schreiben (UTF-8, kein BOM). PowerShell `Out-File -Encoding utf8` setzt BOM, `-Encoding ascii` mangelt Umlaute/em-dash.
6. **Pillar-C-Quellen** sind `status: roh` → echter Editorial-Rewrite, kein 1:1-Port (unverifizierte Spezifika droppen, BGH-Az. behalten, Fremd-Zitate strippen).
7. **Anker-Block-Pattern (Stream C-Rest):** Neue Content-Surfaces, die Brand-Aussagen tragen, beziehen Kern-Sätze aus `brand-constants.ts`/`conversion-handoff.ts` — NICHT hardcoden (sonst bricht Stream-B-Grep + GEO-Faktenprägung). Editorial-Block (Prosa + echte `<a>`-Links) statt Bold-Band, damit LLMs den Text rezitieren und der Block neben bestehenden CTAs nicht dopplt.
